DeepDIVA

In this project we aimed to create a deep-learning synthesizer programmer using the vst plugin DIVA. We wanted to create a tool that addresses the challenge of programming sound synthesizers. This requires a thorough technical understanding as software synthesizers often have many parameters. To tackle this, we built a deep learning model for automatic synthesizer programming that takes a target sound as input, predicts the parameter settings for the synthesizer that causes it to emit as close a sound as possible, and generate a template that you can load in your DIVA synthesizer.
